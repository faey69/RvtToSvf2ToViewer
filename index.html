This is a static webpage demo of using AutoDesk's Model Derivative API, converting rvt file to svf2 and then displaying it using Autodesk Viewer.

Please open the console (CTRL+SHIFT+I) to view various information, such as translation progress, script execution steps, etc.

Please input top to bottom as I skipped most of the validation.

This script was created by referencing and following the steps in the documentation: https://aps.autodesk.com/en/docs/model-derivative/v2/tutorials/prep-roominfo4viewer/

As for the viewer, this documentation was referenced and followed: https://aps.autodesk.com/en/docs/viewer/v7/developers_guide/viewer_basics/starting-html/
